## Command line interface

Menu options view:

```sh
Select options:

> make calculations
  update calculations
  previous calculations
  exit
```

Calculations view:

```sh
Enter Coefficients of Augmented Matrix:

a[1][1] = 12
a[1][2] = 83
a[2][1] = 81
a[2][2] = 91
```

Solutions view:

```sh
Solutions:
a[1] = 14
a[2] = -2
```
